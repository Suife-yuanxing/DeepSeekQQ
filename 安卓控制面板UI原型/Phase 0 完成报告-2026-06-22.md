| 决策 | 内容 |
|------|------|
| **进程隔离** | 8766 API 进程独立于 8082 NoneBot2 进程，通过惰性化 config/db_core/__init__.py 实现 |
| **DB 共享** | 两进程共享 SQLite（WAL 模式），db_core.get_db() 连接池复用 |
